Що таке модуль у Node.js?

Розглядайте модулі так само як і бібліотеки JavaScript.

Набір функцій, які ви хочете включити у свою програму.

Вбудовані модулі

Node.js має набір вбудованих модулів, які ви можете використовувати без подальшої установки.

На офіційному сайті ви зможете знайти та ознайомитись із повним списком модулів.

Включити модулі

Щоб включити модуль, використовуйте функцію require() з назвою модуля:

var http = require('http');

Тепер ваша програма має доступ до модуля HTTP і може створити сервер:

{{ 'http.createServer(function (req, res) { ' }} {{ " res.writeHead(200, {'Content-Type': 'text/plain'}); "}} {{ " res.end('Hello World!'); "}} {{ '}).listen(8080); '}}

Створення власного модуля

Ви можете створити власні модулі та легко включити їх у свої програми.

Наступний приклад створює модуль, який повертає об’єкт дати та часу:

{{ 'exports.myDateTime = function () {' }} {{ " return Date(); "}} {{ '}; '}}

Використовуйте ключове слово exports, щоб зробити властивості та методи доступними поза файлом модуля.

Збережіть наведений вище код у файлі "index.js"

Включити власний модуль

Тепер ви можете включити та використовувати модуль у будь-який із ваших файлів Node.js

index.js

Використовуйте модуль "index" у файлі Node.js:

var http = require('http');

var dt = require('./index');

{{ 'http.createServer(function (req, res) { ' }} {{ " res.writeHead(200, {'Content-Type': 'text/html'}); "}} {{ " res.write('Дата і час зараз: ' + dt.myDateTime()); "}} {{ " res.end(); "}} {{ '}).listen(8080); '}}
Запустити в StackBlitz

Зауважте, що ми використовуємо ./ для пошуку модуля, це означає, що модуль розташований у тій самій теці, що і файл Node.js.

Збережіть наведений вище код у файлі, який називається "new.js", та ініціюйте файл:

Ініціювати new.js:

C:\Users\Your Name>node demo_module.js
