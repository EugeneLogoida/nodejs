Що таке NPM?

NPM — це менеджер пакетів для пакетів Node.js або модулів, які вам потрібно.

www.npmjs.com містить тисячі безкоштовних пакетів для завантаження та використання

Програма NPM автоматично встановлюється на вашому комп’ютері під час інсталяції Node.js

Що таке пакет?

Пакет у Node.js містить усі файли, необхідні для модуля

Модулі — це бібліотеки JavaScript, які можна включити у свій проект.

Завантаження пакету

Завантаження пакету це дуже простий процес для виконання

Відкрийте інтерфейс командного рядка та завантажте потрібний пакет.

Наприклад, можна завантажити пакет під назвою "upper-case":

Завантажити "upper-case":

C:\Users\Your Name>npm install upper-case

Тепер ви завантажили та встановили свій перший пакет!

NPM створює папку з назвою «node_modules», куди буде розміщено пакет. Усі пакети, які ви встановите в майбутньому, будуть розміщені в цій папці.

Тепер проект має таку структуру папок:

C:\Users\My Name\node_modules\upper-case

Використання пакету

Після встановлення пакета він готовий до використання.

Включіть пакет 'upper-case' так само, як ви включаєте будь-який інший модуль:

var uc = require('upper-case');
