Модуль Nodemailer

Модуль Nodemailer дозволяє легко надсилати електронні листи з вашого комп’ютера.

Модуль Nodemailer можна завантажити та встановити за допомогою npm:

Завантажити "nodemailer":

C:\Users\Your Name>npm install nodemailer

Після того, як ви завантажили модуль Nodemailer, ви можете включити його в будь-яку програму:

var nodemailer = require('nodemailer');

Надіслати електронний лист

Тепер ви готові надсилати електронні листи зі свого сервера.

Щоб надіслати електронний лист, використовуйте ім’я користувача та пароль від вибраного постачальника електронної пошти. Цей приклад покаже вам, як використовувати обліковий запис Gmail для надсилання електронних листів:

Приклад

var nodemailer = require('nodemailer');
{{ "var transporter = nodemailer.createTransport({" }} {{ "  service: 'gmail', "}} {{ "  auth: { "}} {{ "   user: 'youremail@gmail.com', "}} {{ "   user: pass: 'yourpassword' "}} {{ "  }"}} {{ "}); "}}
{{ "var mailOptions = {" }} {{ "  from: 'youremail@gmail.com', "}} {{ "  to: 'myfriend@yahoo.com',"}} {{ "  subject: 'Sending Email using Node.js',"}} {{ "  text: 'That was easy!'"}} {{ "}; "}}
{{ "transporter.sendMail(mailOptions, function(error, info){" }} {{ "  if (error) { "}} {{ "   console.log(error); "}} {{ "  } else { "}} {{ "   console.log(error); "}} {{ "  to: 'myfriend@yahoo.com',"}} {{ "   console.log('Email sent: ' + info.response);"}} {{ "  text: 'That was easy!'"}} {{ " }; "}} {{ "}); "}}

І це все! Тепер ваш сервер може надсилати електронні листи.

Кілька приймачів

Щоб надіслати електронний лист кільком одержувачам, додайте їх у властивість 'to' об’єкта mailOptions, розділивши їх комами:

Приклад

{{ "var mailOptions = {" }} {{ "  from: 'youremail@gmail.com', "}} {{ "  to: 'myfriend@yahoo.com','myotherfriend@yahoo.com', "}} {{ "  subject: 'Sending Email using Node.js',"}} {{ "  text: 'That was easy!'"}} {{ "}; "}}

Надіслати HTML

Щоб надіслати текст у форматі HTML у своїй електронній пошті, використовуйте властивість "html" замість властивості "text":

Приклад

{{ "var mailOptions = {" }} {{ "  from: 'youremail@gmail.com', "}} {{ "  to: 'myfriend@yahoo.com','myotherfriend@yahoo.com', "}} {{ "  subject: 'Sending Email using Node.js',"}} {{ "  html: '<h1>Welcome</h1><p>That was easy!</p>'"}} {{ "}; "}}
